1.5 มีอะไรใหม่ๆบ้างที่แตกต่างจาก Version เก่า อย่างน้อย 5 ตัวอย่าง

1.ไม่มี jQuery อีกต่อไป

2.อัพเดท Class และแก้ไข CSS

3.ใช้ icon แบบ svg แทน font icon

4.Containers แบบ Responsive

5.อัพเดท JavaScript แบบครั้งยิ่งใหญ่ เรียกได้ว่าล้างของเก่า

2.จุดเด่้นของ Bootstrap คืออะไร จงอธิบาย

มี UI เริ่มต้นแบบที่สวยงามและใช้งานง่าย

โค๊ดหรือชุดคำสั่งต่าง ๆ ค่อนข้างสะอาดมีโฟลเด้อต้นแบบแค่ 3 ส่วนคือ js, css, fonts

ประหยัดเวลาในการพัฒนาเว็บไซต์และนำไปพัฒนาต่อได้ง่าย

เป็น Responsive Framework พัฒนาเว็บไซต์ที่รองรับการแสดงผลได้หลากหลาย Device

มีการปรับปรุงและพัฒนาอย่างต่อเนื่อง ปัจจุบันเป็นเวอร์ชั่น 3.3.0

ป็นที่นิยมของนักพัฒนาทั่วโลก ทำให้สามารถเรียนรู้และแก้ปัญหาได้ง่าย

จงอธิบายการติดตั้งและใช้งานแบบ CDN และเเบบ Internal (Compiled CSS and JS )แตกต่างกัน อย่างไร

1.เข้าไปที่เว็บ Get bootstrap

2.กดที่ไอคอน ขวาบนรูปตัว Icon bootstrap

3.กดที่คำว่า Read the docs

4.เลื่อนลงไปจนเจอคำว่าCDN Link

5.จะมี 2 ลิ้งคือ css กับ js เลือกว่าจะใช้อันไหน หรือทั้งคู่ก็ได้

6.คัดลอกลิ้งที่จะใช้

7.เข้ามาที่ Vs code แล้ว สร้าง folder ตามปกติ

8.เขียนคำสั่งLink แล้วใส่link ที่คัดลอกมาลงไป

9.เสร็จเรียบร้อย สามารถใช้ Bootstrap ได้แล้ว

การติดตั้ง แบบCDN และ Internal แตกต่างกันที่ CDN เป็นการลิ้งมาจากอินเตอร์เน็ต แต่แบบInternal คือการโหลดไฟล์มาเก็บไว้ในเครื่องและลิ้งจากเครื่องมาอีกที่

ถามไม่มีเน็ต แบบCDN หน้าเว็บมั่วไปหมดเลย แต่แบบ Internal ต่อให้เไม่มีเน็ตก็ยังสามารถRun ได้
